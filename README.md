# FlowSedGen-PINN

AI Framework with Physics-Informed Neural Networks (PINN) for interpretable sediment transport estimation in complex steep channel flows.

## Features

- Physics-informed neural network
- Random Forest
- Support Vector Regression
- Meyer–Peter Müller equation
- SHAP explainability
- Taylor diagram
- Residual analysis
- GUI interface

## Input variables

- So
- Q
- A
- T
- R
- H
- λ/D
- M
- U
- Fr
- Re
- τb
- θ

## Output

- qb
- Φ
- SHAP importance
- model comparison
